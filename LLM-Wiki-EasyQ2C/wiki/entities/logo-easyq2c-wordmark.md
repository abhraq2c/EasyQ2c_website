# EasyQ2C wordmark

## What it is

The primary EasyQ2C wordmark — a text-based logo displaying "EasyQ2C" in two-tone blue typography on a white background.[^logo]

## Properties

- **Type:** brand-asset
- **Owner / Operator:** EasyQ2C Pty Ltd
- **Status in our context:** in-use

### Visual specification

- **Text:** "EasyQ2C" in a single horizontal line, sans-serif, bold, modern.[^logo]
- **Color split:** "Easy" in navy `#2B365C`; "Q2C" in blue `#3B86F4` (sampled from source JPEG).[^logo]
- **SVG font:** Poppins 700 (matches easyq2c.com site font).[^logo]
- **Style:** clean, professional, suitable for headers, favicons, and print.[^logo]

### Usage notes

Prefer **`logo.svg`** or **`logo-transparent-512.png`** for website headers. Use **`logo-transparent-256.png`** as a favicon source. Not currently on the live Zoho homepage (which uses the SDLC diagram). (inferred — intended for new site)
