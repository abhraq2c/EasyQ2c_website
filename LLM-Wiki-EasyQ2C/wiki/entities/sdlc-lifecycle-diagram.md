# SDLC lifecycle diagram

## What it is

A circular software development lifecycle (SDLC) infographic used on the EasyQ2C homepage, illustrating the company's end-to-end delivery process from requirements through support.[^home]

## Properties

- **Type:** brand-asset
- **Owner / Operator:** EasyQ2C Pty Ltd (used on company site; original authorship not stated in source)
- **Status in our context:** in-use

## Description

### Files

- **Live URL:** https://www.easyq2c.com/spd.png[^home]
- **Local copy:** [`raw/assets/easyq2c-sdlc-diagram.png`](../../raw/assets/easyq2c-sdlc-diagram.png)

### Process stages (clockwise)

1. **Requirements** — clipboard/checklist
2. **Analysis and Research** — magnifying glass over charts
3. **Estimate** — cost, effort, time
4. **Design** — vector/pen tool
5. **Development** — laptop with code
6. **Software Testing** — bug inspection
7. **Deployment** — rocket from monitor
8. **Support** — handshake

Central imagery: large light-blue gear with smaller yellow gear; illustrated figure gesturing toward the cycle. Green circular arrow indicates continuous iteration from Support back to Requirements.[^home]

### Website role

Currently serves as the primary visual on the homepage web clip (replacing the wordmark on the live Zoho site). Suitable for "How we work" or services sections.[^home]
